Teste Prático - Desenvolvedor Júnior (PHP/Laravel)
Objetivo

O objetivo deste teste é avaliar suas habilidades em PHP, Laravel, MySQL, HTML, CSS e integração de APIs, especificamente com o Mercado Livre.
Instruções Gerais

    Certifique-se de fornecer instruções claras sobre como rodar seu projeto.
    Você precisará criar uma aplicação no Mercado Livre Developer para obter as credenciais de API (client_id, client_secret e token de acesso).

Desafio 1: Cadastro de Produtos via API do Mercado Livre

Crie um pequeno sistema em Laravel que permita realizar o cadastro de produtos diretamente no Mercado Livre utilizando a API oficial.
Requisitos do Cadastro:

    Um produto deve ter os seguintes atributos:
        Nome
        Descrição
        Preço
        Quantidade em estoque
        Categoria (você pode buscar categorias válidas usando a API de Mercado Livre)
        Imagem (upload de uma imagem que será enviada junto ao produto)

    As seguintes funcionalidades devem ser implementadas:
        Formulário para cadastro de um novo produto.
        Envio das informações do produto para a API do Mercado Livre.
        Exibição da resposta retornada pela API (se o produto foi criado com sucesso ou se houve algum erro).

    Utilize o MySQL para armazenar um histórico dos produtos cadastrados.

    Use HTML e CSS simples para o front-end do formulário.

Desafio 2: Integração com API do Mercado Livre

Para realizar a integração, você deverá utilizar as credenciais de API obtidas no Mercado Livre Developer. 
O fluxo de autenticação OAuth deve ser implementado.
Requisitos da integração:

    Implemente o fluxo OAuth2 para obter o token de acesso ao Mercado Livre.
    O token deve ser utilizado para fazer chamadas autenticadas à API do Mercado Livre, especificamente para o endpoint de criação de produtos (/items).
    Exiba os dados retornados pela API após o produto ser criado (ID do produto, status, etc).

Links úteis:

    Documentação da API do Mercado Livre
    Autenticação OAuth Mercado Livre

Desafio 3: Documentação

Crie um arquivo README.md com:

    Instruções de como instalar e rodar o projeto.
    Explicação sobre como configurar as variáveis de ambiente para a integração com o Mercado Livre (client_id, client_secret, redirect_uri, etc.).
    Passos detalhados de como gerar o token de acesso para autenticação via OAuth.

Bônus (opcional):

    Implementação de um sistema de autenticação no Laravel para proteger o acesso ao formulário de cadastro de produtos.
    Validação de dados no formulário (ex: preços negativos, descrição vazia, etc.).
    Utilização de um framework CSS (ex: Bootstrap ou Tailwind) para melhorar a interface.

O que será avaliado:

    Organização do código.
    Boas práticas em PHP e Laravel.
    Implementação da integração com a API do Mercado Livre.
    Validação e tratamento de erros na comunicação com a API.
    Interface simples e funcional.
